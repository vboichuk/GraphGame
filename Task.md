Мини-игра
* Задано некоторое начальное расположение игровых фишек на топологии (на рисунке они обозначены цветными кругами). 
* Цель игры заключается в том, чтобы перемещением фишек добиться заданного расположения (уменьшенная миниатюра топологии сбоку на экране). 
* Фишки можно перемещать только по существующим путям 
* Фишку невозможно передвинуть сквозь другую фишку на ее пути.
* За один ход фишка может перемещаться через несколько соединений, если указанный пользователем путь свободен от других фишек. 
* Места, куда пользователь может передвинуть выбранную фишку, должны подсвечиваться
* Выбранная фишка должна подсвечиваться.
* Расположение фишек и топология уровня (соединения) должны быть заданы во внешнем текстовом файле. 

Уровень берется из файла './win32-build/bin/GraphGame/Debug/Resources/Levels/level.txt'
(есть также два других файла с уровнями, чтобы игра считала другой уровень необходимо самостоятельно переименовать файлы).

Формат файла с описанием уровня (построчно):
* количество фишек (N);
* количество точек, куда можно поместить фишку (M);
* M строк - координаты точек, куда можно поместить фишку;
* начальное расположение фишек по точкам (в одной строке, через запятую);
* выигрышное положение фишек по точкам (в одной строке, через запятую);
* количество соединений (K);
* K строк - список соединений между парами точек.

Нумерация точек в файле - с единицы.

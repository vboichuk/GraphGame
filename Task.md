Мини-игра
* Задано некоторое расположение игровых фишек (на рисунке они обозначены цветными кругами). 
* Цель игры заключается в том, чтобы перемещением фишек добиться заданного расположения (подложка в виде полупрозрачных цветных кругов). 
* Фишки можно перемещать только по существующим путям 
* Фишку невозможно передвинуть сквозь другую фишку на ее пути.
* За один ход фишка может перемещаться через несколько соединений, если указанный пользователем путь свободен от других фишек. 
* Места, куда пользователь может передвинуть выбранную фишку, должны подсвечиваться
* Выбранная фишка должна подсвечиваться.
* Расположение фишек и топология уровня (соединения) должны быть заданы во внешнем текстовом файле. 

В файле задаются: 
* количество фишек; 
* количество точек, куда можно поместить фишку;
* координаты точек, куда можно поместить фишку;
* начальное расположение фишек по точкам;
* выигрышное положение фишек по точкам;
* количество соединений; 
* список соединений между парами точек.
